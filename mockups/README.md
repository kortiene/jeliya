# Mockups

These are the original product mockups the UI is built to — **layout and
information-architecture reference only, not identity reference**.

Their visual identity predates the brand definition and contradicts it in
places (cube/hexagon logo, multi-hue glowing badges — see the anti-references
in `PRODUCT.md`), and they show the project's former name, Bantaba (renamed
to Jeliya on 2026-07-05 — see `docs/naming.md`). The canonical identity is the flat single-accent
meeting-tree mark: `TreeMark` in `ui/src/components/ui.tsx`,
`ui/public/favicon.svg`, and `assets/banner.svg`.

If a hexagon or a glow from these images shows up in UI chrome, treat it as
a bug.
